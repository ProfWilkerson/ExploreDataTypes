# Explore: What Type of Data Is This?

**Course:** AI Essentials
**Module:** 3
**Type:** Non-graded exploration activity

## Overview
A quick-classification activity presenting 12 short data examples (patient vitals table, call recordings, JSON order log, scanned intake forms, attendance spreadsheet, and more). Students click to classify each example as Structured, Unstructured, or Semi-Structured, then reveal a single answer key at the bottom that provides the correct classification and a brief explanation for each. Prepares students for data-type recognition tasks in the anchor activity and related quiz items.

## File
`index.html`: single self-contained HTML file, no dependencies or external libraries required.

## Instructor Notes
- No student account or login is required.
- The classification buttons and answer key toggle use inline JavaScript; no internet connection is needed for interactivity to function.
- The activity includes 4 structured, 4 unstructured, and 4 semi-structured examples.
- Reflection is self-assessed only. No submission is expected.
